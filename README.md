# 🧧 Chinese Spring Festival / 中国春节 — Stardew Valley Mod

为星露谷物语添加一个全新的节日：**中国春节**（冬季28日）。

Adds a brand-new festival to Stardew Valley: **Chinese Spring Festival** on **Winter 28**.

---

## ✨ 功能 / Features

### 🎆 春节庆典 (Winter 28)
- **节日地点**：鹈鹕镇广场
- **节日时间**：上午 9:00 — 次日 0:00
- **烟花表演**：与镇长Lewis对话触发壮观的烟花秀
- **全NPC对话**：30+ 位NPC都有独特的春节主题对话
- **邮件通知**：冬季25日收到Lewis镇长的春节邀请函
- **双语支持**：根据游戏语言自动切换中/英文

### 🧨 NPC 互动
所有主要村民都会出现在节日现场，每人都有精心编写的节日对话，包含对春节文化的独特反应和感受。

---

## 📦 安装 / Installation

### 前置要求
1. [SMAPI](https://smapi.io/) — 星露谷物语Mod加载器（v4.0.0+）
2. [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915) — 内容补丁框架（v2.0.0+）

### 安装步骤
1. 安装 SMAPI 和 Content Patcher（如果尚未安装）
2. 将本 Mod 文件夹 `StardewValley-SpringFestival` 放入游戏的 `Mods` 目录
3. 启动游戏，等到冬季28日即可体验春节！
